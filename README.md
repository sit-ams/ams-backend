# AMS - backend

## ENVIROMENT VARIABLES

| Key         | Value                  |
| ----------- | ---------------------- |
| PORT        | 8000                   |
| NODE_ENV    | development/production |
| DB_USERNAME | `database username`    |
| DB_PASSWORD | `database password`    |
| DB_NAME     | `database name`        |
| DB_HOST     | `database host`        |
| JWT_SECRET  | `JWT secret key`       |
